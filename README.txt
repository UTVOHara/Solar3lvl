Поздравлятор 3lvl
Веб-приложение для учета дней рождений, реализованное на серверной части с использованием ASP.NET Core MVC. Для хранения информации о днях рождения используется база данных. Главная страница приложения отображает список именинников, у которых сегодня или в ближайшем будущем день рождения. Дополнительный функционал доступен на отдельных страницах, ссылки на которые можно найти на главной странице. Приложение также позволяет сохранять и отображать фотографии именинников."Поздравлятор" - это веб-приложение для учета дней рождений, реализованное на серверной части с использованием ASP.NET Core MVC.
